Declaration Translations

Manage the source-language declaration that appears below the English declaration on the certificate (e.g. the Chinese version for Chinese→English translations). A translation is matched to a document by its source language name (must exactly match the certified-language "from" name, e.g. Chinese).

These fields may contain placeholder tokens that get filled in per document: {fullName}, {memberNumber}, {fromLanguage}, {toLanguage}, {date}, {documents}.
